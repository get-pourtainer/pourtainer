## Features

### CONTAINERS
- Start, Stop, Restart, Kill, and Remove containers
- View container logs in real-time
- Inspect container details and stats
- Attach to running containers

### STACKS
- Deploy and manage Docker Compose stacks
- View stack services and status
- Pull and redeploy stacks

### IMAGES
- Browse and pull images
- Remove unused images
- View image layers and history

### NETWORKS & VOLUMES
- Create and manage networks
- Inspect volume details
- Connect/disconnect containers

### ENVIRONMENTS
- Multiple Portainer environment support
- Quick environment switching
- Edge agent support

## Environment
- macOS 14+ (Sonoma)
- Node v24
- Npm v10.9 (please don't commit a `yarn.lock` file, we are using `npm` both for development and building)
- Xcode 26
